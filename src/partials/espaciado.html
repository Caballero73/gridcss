Espaciado entre filas y columnas

Por defecto, la grilla tiene sus celdas pegadas. Aunque es posible darle un margen a las celdas con la propiedad margin, existe una forma más adecuada, que evita problemas con el modelos de caja.

Para definir el espacio entre celdas, utilizamos las propiedades grid-column-gap y/o grid-row-gap.


Propiedad Descripción
grid-column-gap Establece el tamaño de los huecos entre columnas (líneas verticales).
grid-row-gap Establece el tamaño de los huecos entre filas (líneas horizontales).

Con ellas indicaremos el tamaño de dichos espacios.


      HTML
      <div class="padre">
        <div class="hijo">Contenido... #1</div>
        <div class="hijo">Contenido... #2</div>
        <div class="hijo">Contenido... #3</div>
        <div class="hijo">Contenido... #4</div>
        <div class="hijo">Contenido... #5</div>        
        <div class="hijo">Contenido... #6</div>
        <div class="hijo">Contenido... #7</div>
        <div class="hijo">Contenido... #8</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% 50% 30%;
        grid-template-rows: auto;
        grid-column-gap: 3rem;
        grid-row-gap: 1rem; 
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }

Existe una forma abreviada para las propiedades grid-column-gap y grid-row-gap y no tener que indicarlas a cada una. La propiedad sería grid-gap y se utiliza de la siguiente forma:

      grid-gap: Fila / Columna;
      grid-gap: 1rem 3rem;

lo cual equivale a

      grid-row-gap: 1rem;
      grid-column-gap: 3rem;
