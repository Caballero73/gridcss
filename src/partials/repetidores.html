Repetidores, unidades de medida y funciones

• repeat() ... en las propiedades grid-template-columns y grid-template-rows podemos indicar expresiones de repetición, indicando celdas que repiten un mismo patrón de celdas varias veces. La expresión a utilizar sería la siguiente: repeat("núm de veces", "valor o valores").

      HTML
      <div class="padre"> ◀ Este es el "Grid Container"
        <div class="hijo">Contenido... #1</div> ◀ Estos son los "Grid Item"
        . . .
        <div class="hijo">Contenido... #12</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% repeat(2, 40%);
        grid-template-rows: repeat(2, 5rem 10rem); ◀ Uso de "repeat"
        grid-gap: .5rem;
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }

• minmax () define un rango de tamaño mayor o igual al min (mínimo) y menor o igual que el max (máximo). Veamos en siguiente ejemplo:

      .padre {
        display: grid;
        grid-template-columns: minmax(200px, 33.333%) minmax(10px, 33.333%) minmax(10px, 33.333%);
        grid-template-rows: auto;
        grid-gap: .5rem;
      }

Con esto tenemos una grilla de filas que se ajustan automáticamente al contenido y tres columnas, donde la primera tendrá un ancho mínimo de 200px y un máximo de 33.333%, las otras dos tendrán un mínimo de 10px y un máximo de 33.333%. Esto lógicamente se nota al disminuir el ancho disponible en la pantalla.

Podemos combinar minmax con repeat(), obteniendo el mismo resultado:

      .padre {
        display: grid;
        grid-template-columns: minmax(200px, 33.333%) repeat(2, minmax(10px, 33.333%));
        grid-template-rows: auto;
        grid-gap: .5rem;
      }

• Unidades de medida ... CSS Grid permite la utilización de distintas unidades (tales como px, rem, em y/o %) para definir anchos, altos y espaciados, incluso, dentro de una misma propiedad.

Sin embargo crear una grilla utilizando sólo medidas estáticas como pixeles puede resultar poco flexible. CSS Grid otorga una nueva unidad de medida llamada fr, alias de fracción.

Veamos en el siguiente ejemplo como podemos usarla:

      .padre {
        display: grid;
        grid-template-columns: 1fr 2fr .5fr;
        grid-template-rows: auto;
        grid-gap: .5rem;
      }
