Grids identados

Simplemente lo que debemos hacer es colocar en la celda seleccionada los elementos que deseamos anidar.

      HTML
      <div class="padre">
        <div class="hijo">Contenido... #1</div>
        <div class="hijo">Contenido... #2</div>
        <div class="hijo">Contenido... #3</div>
        <div class="hijo">Contenido... #4</div>
        <div class="hijo">Contenido... #5
          <div class="hijo">Sub-Contenido... #1</div> ◀ Grilla indentada
          <div class="hijo">Sub-Contenido... #2</div>
          <div class="hijo">Sub-Contenido... #3</div>
        </div>
        <div class="hijo">Contenido... #6</div>
        <div class="hijo">Contenido... #7</div>
        <div class="hijo">Contenido... #8</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% 50% 30%;
        grid-template-rows: auto; 
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }

      .hijo:nth-of-type(5) {
        background-color: rgb(182, 202, 226);
        display: grid; ◀ Esto permite indentar la grilla
      }
