Manejando el grid implicito

El Grid Implícito son todas aquellas celdas cuyas características no han sido predefinidas.

Podemos las propiedades grid-auto-columns y grid-auto-rows para darle un tamaño automático a las celdas de la cuadrícula, sin importar cuantas definamos en un momento X. Para ello, sólo hay que especificar el tamaño deseado en cada una de las propiedades.

Adicionalmente, utilizaremos grid-auto-flow para señalar el flujo de elementos en la cuadrícula.


Propiedad Valores Descripción
grid-auto-columns [tamaño] Indica el tamaño de ancho que tendrán las columnas automáticas.
grid-auto-rows [tamaño] Indica el tamaño de alto que tendrán las filas automáticas.
grid-auto-flow row | column | dense Utiliza un algoritmo de autocolocación (intenta rellenar los espacios vacíos).

Ejemplo 1

      HTML
      <div class="padre">
        <div class="hijo"></div>
        /...  .../
        <div class="hijo a">Soy una celda automática</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: [inicio-col] 37% [segunda-col] 
        25% [tercer-col] 15% [cuarta-col] 15% [final-col];
        grid-template-rows: [inicio-fila] 12rem [segunda-fila]
        12rem [final-fila];
        grid-gap: 3px;
        grid-auto-flow: row; ◀ La celdas fluyen en filas
      }
      
      .hijo {text-align: center;}
      
      .hijo:nth-of-type(1) {grid-column: inicio-col / segunda-col;
        grid-row: inicio-fila / final-fila;}

      .hijo:nth-of-type(2) {grid-column: segunda-col / tercer-col;
        grid-row: inicio-fila / final-fila;}

      .hijo:nth-of-type(3) {grid-column: tercer-col / final-col;
        grid-row: inicio-fila / segunda-fila;}

      .hijo:nth-of-type(4) {grid-column: tercer-col / cuarta-col;
        grid-row: segunda-fila / final-fila;}

      .hijo:nth-of-type(5) {grid-column: cuarta-col / final-col;
        grid-row: segunda-fila / final-fila;}

grid-auto-flow acepta una palabra reservada para describir el planteamiento de "cargado". Por defecto éste valor es sparse, pero podemos alterar ésto a dense que intenta cubrir todas los huecos disponibles. No hace falta que coloquemos la alabra row por que ya está implícita.

        grid-auto-flow: dense; ◀ La celdas fluyen mas densamente
