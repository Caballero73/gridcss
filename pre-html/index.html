CSS Grid Layout

CSS Grid Layout es el sistema de diseño más poderoso disponible en CSS. Es un sistema bidimensional, lo que significa que puede manejar tanto columnas como filas, a diferencia de flexbox, que es en gran medida un sistema de 1 dimensión.

Cuando trabajas con Grid Layout aplicas reglas CSS tanto para un elemento padre (que se convierte en el Contenedor Grid) como para los elementos hijos (que se convierten en Grid Items).

Conceptos fundamentales

• Grid Container es el elemento "Padre" que tiene un nuevo tipo de display, display: grid;. Nos permite colocar otras propiedades para manipular nuestro layout.

• Grid Item son nuestro componentes, contenido, lo que vamos a manejar. Nuestras filas o columnas que vamos a mover a nuestro gusto. Es importante tomer en cuenta que son hijos directos de "Grid Container".

      <div class="wrapper"> ◀ Este es el "Grid Container"
        <div class="one">Uno</div> ◀ Estos son los "Grid Item"
        <div class="two">Dos</div>

        <div class="four">
            Cuatro
            <p>Sub-item</p> ◀ Este no es un "Grid Item"
        </div>
        <div class="five">Cinco</div>
        <div class="six">Seis</div>
      </div>

• Grid Line son las líneas divisorias horizontales y verticales. Se crean cuando se definen las columnas y filas del layout.

• Grid Track Espacio entre dos líneas adyacentes. mejor conocidos como "Filas y columnas".

• Grid Cell mejor conocidas como "Celdas", es el espacio entre dos filas adyacentes y dos columnas adyacentes.

• Grid Area espacio rodeado o delimitado por 4 grid lines.

Definiendo columnas

Para utilizar cuadriculas Grid CSS, lo primero que debemos definir es lo siguiente:

• display: grid   ó   display: inline-grid ... para activar la cuadrícula grid hay que utilizar sobre el elemento contenedor (padre) la propiedad display y especificar el valor grid o inline-grid. Este valor influye en como se comportará la cuadrícula con respecto al contenido exterior.

El primero, grid permite que la cuadrícula aparezca encima/debajo del contenido exterior (en bloque) y el segundo de ellos, inline-grid permite que la cuadrícula aparezca a la izquierda/derecha (en línea) del contenido exterior.

Elemento Descripción
 inline-grid  Establece una cuadrícula con ítems en línea, de forma equivalente a inline-block.
 grid  Establece una cuadrícula con ítems en bloque, de forma equivalente a block.

Una vez definido uno de estos dos valores para la propiedad display del elemento contenedor, hay varias formas de configurar nuestra cuadrícula grid. Comencemos con las propiedades que se aplican al elemento contenedor (padre) para definir las columnas.


• grid-template-columns establece el tamaño de las columnas (eje horizontal).

En este ejemplo he utilizado porcentajes como unidad, sin embargo, también podemos utilizar otras unidades (e incluso combinarlas) como píxeles, em, rem, la palabra clave auto (que obtiene el tamaño restante) o la unidad especial fr (fraction), que simboliza una fracción de espacio restante en el grid, y la cual se expliacrá mejor más adelante.

        <div class="hijo">Contenido... #10</div>
        <div class="hijo">Contenido... #11</div>
        <div class="hijo">Contenido... #12</div>

Definiendo filas

Una vez definidos el tipo de display y las columnas, debemos completar la estructura básica de la grilla con las filas (eje vertical).


• grid-template-rows establece el tamaño de las filas (eje vertical).

Combinado las columnas y las filas en el siguiente ejemplo tenemos una cuadricula con 3 columnas (la primera con 20% de ancho, la segunda con 50% de ancho y la tercera con 30% de ancho) y con 3 filas (la primera con 15rem de alto, la segunda con 5rem de alto y la tercera con 10rem). Ahora, de acuerdo al número de ítems (elementos hijos) que tiene el contenedor grid, tenemos una cuadrícula de 3x3 elementos. Como el número de ítems es par, la última celda de la cuadrícula se queda vacía.

      HTML
      <div class="padre">
        <div class="hijo">Contenido... #1</div>
        <div class="hijo">Contenido... #2</div>
        <div class="hijo">Contenido... #3</div>
        <div class="hijo">Contenido... #4</div>
        <div class="hijo">Contenido... #5</div>
        <div class="hijo">Contenido... #6</div>
        <div class="hijo">Contenido... #7</div>
        <div class="hijo">Contenido... #8</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% 50% 30%;
        grid-template-rows: 15rem 5rem 10rem; ◀ Esto define las filas
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;

Espaciado entre filas y columnas

Por defecto, la grilla tiene sus celdas pegadas. Aunque es posible darle un margen a las celdas con la propiedad margin, existe una forma más adecuada, que evita problemas con el modelos de caja.

Para definir el espacio entre celdas, utilizamos las propiedades grid-column-gap y/o grid-row-gap.


Propiedad Descripción
grid-column-gap Establece el tamaño de los huecos entre columnas (líneas verticales).
grid-row-gap Establece el tamaño de los huecos entre filas (líneas horizontales).

Con ellas indicaremos el tamaño de dichos espacios.


      HTML
      <div class="padre">
        <div class="hijo">Contenido... #1</div>
        <div class="hijo">Contenido... #2</div>
        <div class="hijo">Contenido... #3</div>
        <div class="hijo">Contenido... #4</div>
        <div class="hijo">Contenido... #5</div>        
        <div class="hijo">Contenido... #6</div>
        <div class="hijo">Contenido... #7</div>
        <div class="hijo">Contenido... #8</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% 50% 30%;
        grid-template-rows: auto;
        grid-column-gap: 3rem;
        grid-row-gap: 1rem; 
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }

Existe una forma abreviada para las propiedades grid-column-gap y grid-row-gap y no tener que indicarlas a cada una. La propiedad sería grid-gap y se utiliza de la siguiente forma:

      grid-gap: Fila / Columna;

      grid-row-gap: 1rem;
      grid-column-gap: 3rem;

Repetidores, unidades de medida y funciones

• repeat() ... en las propiedades grid-template-columns y grid-template-rows podemos indicar expresiones de repetición, indicando celdas que repiten un mismo patrón de celdas varias veces. La expresión a utilizar sería la siguiente: repeat("núm de veces", "valor o valores").

      HTML
      <div class="padre"> ◀ Este es el "Grid Container"
        <div class="hijo">Contenido... #1</div> ◀ Estos son los "Grid Item"
        . . .
        <div class="hijo">Contenido... #12</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 20% repeat(2, 40%);
        grid-template-rows: repeat(2, 5rem 10rem); ◀ Uso de "repeat"
        grid-gap: .5rem;
      }

      .hijo {
        background-color: rgb(255, 243, 204);
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }

      .padre {
        display: grid;
        grid-template-columns: minmax(200px, 33.333%) repeat(2, minmax(10px, 33.333%));
        grid-template-rows: auto;
        grid-gap: .5rem;
      }

• Unidades de medida ... CSS Grid permite la utilización de distintas unidades (tales como px, rem, em y/o %) para definir anchos, altos y espaciados, incluso, dentro de una misma propiedad.

Sin embargo crear una grilla utilizando sólo medidas estáticas como pixeles puede resultar poco flexible. CSS Grid otorga una nueva unidad de medida llamada fr, alias de fracción.

Veamos en el siguiente ejemplo como podemos usarla:

      .padre {
        display: grid;
        grid-template-columns: 1fr 2fr .5fr;
        grid-template-rows: auto;
        grid-gap: .5rem;
      }

Definiendo areas de contenido

por lo general nuestra web no será una simple grilla de columnas y filas en forma de "cuadrícula" o tablero de ajedrez... por eso es que Grids CSS permite indicar el nombre y posición concreta de cada área de una cuadrícula. Utilizando la propiedad grid-template-areas, para especificar el orden de las distintas áreas en la cuadrícula, y luego en cada ítem hijo, usamos la propiedad grid-area para indicar el nombre del área en específico.

Propiedad Descripción
grid-template-areas Indica la disposición de las áreas en el grid. Cada texto entre comillas simboliza una fila.
grid-area Indica el nombre del área. Se usa sobre ítems hijos del grid.

De esta manera, es muy fácil crear una grilla totalmente personalizada con sólo unas cuantas líneas de CSS:

      
      HTML
      <div class="padre mt-2">
        <div class="hijo head">Cabecera</div> 
        <div class="hijo aside">Aside</div>
        <div class="hijo main">Contenido</div>
        <div class="hijo footer">Footer o pié</div>
      </div>

      CSS
      .padre {
        display: grid;
        grid-template-columns: 1fr 3fr;
        grid-template-rows: 1fr 8fr 1fr;
        grid-template-areas: "head head" 
                             "aside main" 
                             "footer footer";
        height: 70vh;
      }

      .hijo {
        border: 1px solid #777;
        padding: 10px;
        text-align: center;
      }
      ◀ cada item debe tener una clase propia
      .head {
          grid-area: head;
          color: #f8f8ff;
          background: #09458d;
      }
      .aside {
        grid-area: aside;
        background: #ff3232;
      }
      .main {
        grid-area: main;
        background: #518f1e;
      }
      .footer {
        grid-area: footer;
        background: #ffc300;
      }

En la propiedad grid-template-areas, también podemos indicar una palabra clave en lugar de indicar el nombre del área:

  • La palabra clave none: Indica que no se colocará ninguna celda en esta posición.
  • Uno o más puntos (.): Indica que se colocará una celda vacía en esta posición.
